En **moins de 10 minutes**, vous allez créer votre compte, monter votre premier espace de travail et inviter vos collaborateurs. Suivez le guide.

:::video src="oJmNRXcqr18" title="Tutoriel : créer son premier compte (5 min)" /:::

## Avant de commencer

Préparez :

- Une **adresse email** valide (vous recevrez les notifications dessus)
- Le **logo** de votre entreprise (optionnel, format PNG ou SVG, fond transparent de préférence)
- Quelques infos de base : nom de l'entreprise, ville, devise utilisée

:::callout type="tip" title="Pas de carte bancaire requise"
Vous pouvez utiliser LouraTech gratuitement le temps de tester. Aucune information de paiement n'est demandée à l'inscription.
:::

## 1. Créer votre compte

Rendez-vous sur la page d'inscription et remplissez le formulaire. C'est très simple :

:::image src="/images/docs/register-form.png" alt="Page d'inscription LouraTech" caption="Le formulaire d'inscription — sobre, rapide, sans champs inutiles." /:::

Pour le mot de passe, retenez :

- **8 caractères minimum**
- Au moins **1 majuscule**
- Au moins **1 chiffre**
- Une force "Bon" ou "Excellent" est recommandée

:::callout type="warning" title="Choisissez un mot de passe fort"
Votre compte donne accès à toutes vos données : équipe, ventes, clients. Évitez les mots de passe simples comme votre date de naissance ou "azerty1234". Un gestionnaire de mots de passe (Bitwarden, 1Password) est fortement conseillé.
:::

Validez le formulaire. Une page de bienvenue animée s'affiche pendant que votre espace se prépare, puis vous arrivez sur le **tableau de bord**.

## 2. Créer votre première organisation

L'**organisation** est l'espace de travail de votre entreprise. Tout (employés, ventes, produits) y sera rattaché.

:::callout type="info" title="Une visite guidée vous attend"
Dès votre arrivée sur le tableau de bord, une visite interactive vous montre où cliquer. Suivez-la — elle ne dure que 30 secondes et vous fait gagner beaucoup de temps.
:::

Cliquez sur **Créer maintenant** et laissez-vous guider à travers les 4 étapes :

:::steps

1. **Donnez un nom à votre organisation**

   Mettez le nom commercial sous lequel vos clients vous connaissent. Vous pourrez le modifier plus tard si besoin.

2. **Choisissez une catégorie**

   Sélectionnez votre secteur d'activité (commerce, restauration, services, etc.). Cela alimentera vos rapports automatiquement.

3. **Configurez les paramètres**

   Indiquez votre **pays** et votre **devise**. La devise sera utilisée par défaut pour toutes vos ventes et factures.

4. **Ajoutez votre logo**

   Glissez votre fichier image dans la zone prévue. Le logo apparaîtra sur toutes vos factures, contrats et bulletins.
   :::

:::image src="/images/docs/create-org-step-2.png" alt="Étape 2 de la création d'organisation" caption="L'étape 'Catégorie' — sélection visuelle parmi les secteurs disponibles." /:::

À la finalisation, vous arrivez sur le **tableau de bord de votre organisation**. Une nouvelle visite guidée se lance pour vous présenter la barre latérale et les modules.

## 3. Personnaliser votre branding

Avant d'émettre vos premières factures, prenez 5 minutes pour configurer votre **branding**. C'est ce qui rendra vos documents pros.

Allez dans `Paramètres → Branding`. Renseignez :

- **Adresse postale complète** — apparaît en pied de chaque document
- **Identifiant fiscal / RCCM** — obligatoire sur les factures dans la plupart des pays
- **Mentions légales** — texte libre (conditions de paiement, garanties, etc.)
- **Couleur d'accent** — utilisée sur les en-têtes de documents

:::video src="oJmNRXcqr18" title="Configurer son branding (3 min)" /:::

:::callout type="success" title="Vous économiserez du temps"
Configurer le branding une fois pour toutes évite des dizaines de modifications manuelles plus tard. Toutes vos factures, devis et contrats seront automatiquement à votre image.
:::

## 4. Inviter votre équipe

Place aux humains. Allez dans `Ressources humaines → Employés` et cliquez sur **Inviter**.

Pour chaque membre :

:::steps

1. **Saisissez son email**

   Le membre recevra un lien d'invitation valide 7 jours.

2. **Choisissez son rôle**

   Le rôle détermine ce qu'il peut faire : Admin (tout), RH (employés/paie), Vendeur (caisse uniquement)…

3. **Optionnel : département et poste**

   Pour structurer votre organigramme dès le début. Vous pourrez modifier plus tard.

4. **Ajoutez un mot d'accueil**

   Personnalisez l'email d'invitation — un petit message fait toujours plaisir.
   :::

:::image src="/images/docs/invite-member.png" alt="Formulaire d'invitation" caption="Inviter un membre — email + rôle suffisent pour démarrer." /:::

:::callout type="tip" title="Invitez en plusieurs fois"
Pas besoin de tout faire d'un coup. Commencez par 1 ou 2 personnes clés, vérifiez que tout fonctionne, puis élargissez.
:::

## 5. Votre première vente

Si vous gérez du stock, voici la séquence minimale pour encaisser votre première vente :

:::steps

1. **Créez un entrepôt**

   `Stocks → Entrepôts → Nouveau`. Mettez "Boutique" ou "Stock principal" si vous n'avez qu'un site.

2. **Ajoutez vos produits**

   `Stocks → Produits → Nouveau`. Saisissez le nom, le prix de vente, et activez le suivi de stock pour les biens physiques.

3. **Saisissez le stock initial**

   `Stocks → Mouvements → Nouveau`. Choisissez "Entrée", l'entrepôt, le produit, la quantité de départ.

4. **Encaissez via la caisse**

   `Stocks → Caisse`. Cliquez sur le produit, ajustez la quantité, choisissez le client et la méthode de paiement, validez.
   :::

:::callout type="success" title="Bravo, votre première facture est prête"
Vous pouvez la télécharger en PDF, l'envoyer par email au client, ou simplement la conserver pour vos archives. Le stock est automatiquement décrémenté.
:::
